## 英语部分

### 获取单词

> 可以指定每页容纳多少单词，并且是第几页

### url

/english/word/get

### 接口方法

GET

### 传入参数

```json
{
    "openid": "12hsx65cfx",
    "number_every_page": 20,         // 每页多少单词
    "page_offset": 0                 // 从0开始的页数偏移量，0代表第一页

}
```

### 返回参数

```json
{
    "code": 0,                       // 0-成功 其他表示失败，错误码对应具体错误
    "data": [
        {
            "id": 1,
            "english": "rat",
            "chinese": "老鼠"
        },
        {
            "id": 2,
            "english": "valley",
            "chinese": "山谷"
        },
        ...                          // 根据传入参数，data 列表的长度应该是20
    ]

}
```
